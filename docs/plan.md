# AIGTD 开发计划

## 项目概述
基于PRD开发一款智能任务管理助手，采用"思考 -> 判断 -> 反馈 -> 推进"的闭环模式。

## 技术栈
- 前端：React + Next.js  
- 后端：Python Flask
- 数据库：Supabase + 本地SQLite
